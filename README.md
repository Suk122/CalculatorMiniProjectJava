## Java Calculator Application

## Description

This is a simple **console-based calculator application** developed using Java.
It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Continuous calculation using loop 
* Handles division by zero 
* User-friendly menu system 

## Technologies Used

* Java
* Scanner (for user input)

## Project Structure

Calculator.java

## How to Run

1. Open the project in your IDE (IntelliJ / VS Code)
2. Compile the program:

```
javac Calculator.java
```

3. Run the program:

```
java Calculator
